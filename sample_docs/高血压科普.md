# 高血压防治科普

原发性高血压是一种常见的慢性心血管疾病，就诊科室为心血管内科。患者常见的症状包括头痛、头晕、心悸、颈项板紧和疲劳。

该病的主要病因与高钠饮食、超重肥胖、长期精神紧张以及过量饮酒有关。

在诊断方面，医生通常会进行诊室血压测量，并结合24小时动态血压监测来评估全天血压波动。诊室血压测量需要使用电子血压计，动态血压监测则使用动态血压监测仪。

治疗上以生活方式干预为基础，包括限盐、减重和规律运动；当血压控制不佳时采用药物降压治疗，常用药物包括钙拮抗剂和ACEI类药物。

需要长期监测的临床指标包括收缩压和舒张压。若长期控制不佳，高血压可能并发2型糖尿病和慢性肾脏病。

# 糖尿病基础知识

2型糖尿病是一种内分泌代谢疾病，就诊科室为内分泌科。常见症状有多饮、多尿、多食、体重下降和视物模糊。其病因主要是胰岛素抵抗叠加肥胖与缺乏运动。

诊断步骤包括空腹血糖检测、口服葡萄糖耐量试验和糖化血红蛋白检测，其中空腹血糖检测使用血糖仪。治疗包括医学营养与运动治疗、口服降糖药物以及必要时的胰岛素治疗。需要监测的临床指标为空腹血糖和糖化血红蛋白。
